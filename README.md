# zh-sport-hub

## 简介

zh-sport-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态页面资源的存储与展示空间。

页面内容涵盖体育相关主题的独立信息页面，所有页面均为纯 HTML 实现，便于直接访问与分享。

## 目录结构

```
.
├── README.md           # 本文件
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放归档的 HTML 页面
│   ├── example1.html   # 示例页面 1
│   ├── example2.html   # 示例页面 2
│   └── ...
└── assets/             # 资源文件（样式、图片等）
    ├── styles/
    └── images/
```

- `pages/`：主要页面存放目录，每个文件为一个独立页面。
- `assets/`：页面所需的公共资源文件，按类型分目录存放。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独打开使用，无需依赖其他服务。
- 页面内容以体育信息为主，不包含动态后端逻辑。
- 页面命名建议使用英文或拼音，避免特殊字符。
- 归档页面可能不定期更新或移除，以保持内容相关性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接双击 `pages/` 目录下的 HTML 文件，即可在浏览器中查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
